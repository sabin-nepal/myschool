
# My School

A brief description of what this project does and who it's for

Our My School Theme is for the school project .
## Installation

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in My School in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.


## Documentation

1. For changing logo
=> Go to appearance >> customize >> Site Identity >> logo

2. Different theme options
=> Appearance >> customize >> theme options

3. Changes Colors
=> Appearance >> customize >> Colors

4. Use social Links
=> Appearance >> Customize >> Social Links
